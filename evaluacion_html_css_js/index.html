Sobre esta evaluación

Esta evaluación combina las tres grandes áreas del desarrollo web: estructura (HTML), estilo (CSS) e interactividad (JavaScript), integradas con el framework Bootstrap. A través de este proyecto demostramos nuestra capacidad para construir una web moderna, adaptable y dinámica.

HTML

Es la base de toda página web. Nos permite estructurar el contenido y definir las secciones que luego serán estilizadas con CSS y dinamizadas con JavaScript.

CSS

El lenguaje que define el estilo visual del sitio: colores, tamaños, fuentes y disposición. Bootstrap ayuda a crear interfaces modernas fácilmente.

JavaScript

El motor de la interactividad. Permite responder a las acciones del usuario, modificar el contenido en tiempo real y generar experiencias dinámicas.

Interactividad con JavaScript

En esta sección podés interactuar con un botón que cambia el color de fondo. Esto demuestra cómo los eventos de JavaScript pueden modificar el comportamiento y estilo de una página en tiempo real.

Ventajas de usar JavaScript
  • Agrega interactividad a las páginas web
  • Permite validar formularios sin recargar la página
  • Facilita la creación de animaciones y efectos visuales
  • Trabaja junto con HTML y CSS para crear experiencias completas
